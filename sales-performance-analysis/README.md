# Sales Performance Analysis

## Objective
Analysis performed to identify sales trends from 2015-2018.

## Dataset
Dataset obtained from [Kaggle](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting).

## Tools Used
* Jupyter Notebook
* Python
* numpy
* pandas
* matplotlib
* seaborn
* Tableau

## Process
* Data cleaning
  * Dataset imported, cleaned, and initially analyzed using Jupyter Notebook and a combination of Puthon libraries
* Analysis
  * Initial evaluation and analysis performed while cleaning and normalizing the dataset in Jupyter Notebook
  * Final analysis and visualizations performed using Tableau

## Results / Insights
* Sales continue to increase each year
  * The exception being that the first quarter of each calendar year sees a dramatic drop in performance
    * 2016 Q1: -64.87%
    * 2017 Q1: -48.74%
    * 2018 Q1: -47.84%
* The West region sees the highest revenue with only one exception
  * 2016 - Top Region - East
* The Consumer segment is by far the largest contributor to sales
* The Technology category, specifically phones, are the largest source of revenue
  * However, the sales of phones is extremely volatile and is only a strong seller during specific quarters
    * 2015 Q2 and Q4
    * 2016 Q3
    * 2018 Q3 and Q4
* Key Insights
  * Marketing toward consumers in the West region will likely see a high return on investment
  * Targetting chairs along with the latest phone releases in Q3 and Q4 will help ensure profitability
